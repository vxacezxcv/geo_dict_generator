# README


* area.sql:						Geo SQL database
* db_dumper.sh:					script for dumping database to text files.
* db\_dump_provinces.txt:		province information
* db\_dump_cities.txt:			city information
* db\_dump_areas.txt:			area information
* create\_geo_dict.py:			further process the dumped text files.
* geo_dict.py:					location py-dict
* geo\_dict_short.py:			location py-dict, with short names
* geo\_dict\_short_jieba.py:	location list for jieba segmentation
* ethnic_dict.txt:				ethnic list for shorting geo names
